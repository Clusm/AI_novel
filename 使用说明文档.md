# AI 小说写作系统 - 使用说明文档

本文档详细介绍了 AI 小说写作系统的安装、配置及使用方法。本系统基于 DeepSeek、通义千问等大模型，提供智能化的长篇小说辅助写作功能。

## 1. 安装与启动

### 1.1 安装
1.  **获取安装包**：找到 `AI_Novel_Writer_Setup_v2.4.exe` 安装程序。
2.  **运行安装**：双击运行，按照提示完成安装。
    *   建议保持默认安装路径，或安装在非系统保护目录（如 D盘）。

### 1.2 启动
*   双击桌面上的 **AI Novel Writer Pro** 图标即可启动。
*   启动后会出现一个黑色终端窗口（用于显示运行日志，请勿关闭）和一个图形化操作界面。

---

## 2. 系统配置

首次使用前，需要配置 AI 模型的 API Key。

### 2.1 API 配置
1.  点击界面左上角的 **"设置"** 按钮（或在欢迎页点击配置）。
2.  在弹出的配置窗口中填写以下信息：
    *   **DeepSeek Key**：用于深度推理和大纲分析。
    *   **通义千问 Key**：用于主要内容的生成和记忆检索。
    *   **Kimi Key**（可选）：用于特定风格的润色。
3.  **路由策略**：
    *   `speed`（极速）：优先使用响应最快的模型。
    *   `quality`（质量）：优先使用推理能力更强的模型（推荐）。
    *   `balanced`（平衡）：在速度和质量之间折中。
4.  **主写模型**：
    *   `auto`（自动）：根据路由策略自动选择。
    *   `qwen`（通义千问）：速度快、稳定性高，适合大批量生成。
    *   `kimi`（Kimi）：文风细腻、逻辑性强，但生成速度较慢（需要配置 Kimi Key）。
5.  点击 **"保存配置并验证"**，系统会自动测试连接。

### 2.2 模型组合策略详解

系统会根据你选择的 **路由策略** 和 **主写模型** 自动组合不同的 AI 模型，以达到最佳效果：

| 路由策略 | 大纲优化 (DeepSeek) | 设定守护 (Qwen) | 主写手 (Writer) | 润色审校 (Editor) | 特点 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Speed (极速)** | deepseek-chat | qwen-plus | qwen-plus | qwen-plus | **速度最快**，成本最低，适合快速出稿。 |
| **Balanced (平衡)** | deepseek-chat | qwen-plus | qwen-plus | **kimi-k2.5** | 兼顾速度，**润色环节**使用 Kimi 提升文笔。 |
| **Quality (质量)** | **deepseek-reasoner (R1)** | **qwen-max** | **kimi-k2.5** | **kimi-k2.5** | **质量最优**，使用深度思考模型和大参数模型，文笔细腻，但生成较慢。 |

> **注意**：
> *   如果未配置 Kimi Key，系统会自动回退使用 Qwen 模型。
> *   DeepSeek R1 (reasoner) 模型思考时间较长，请耐心等待。

### 2.3 数据存储位置
为了防止数据丢失和权限问题，本系统的数据存储遵循以下规范：
*   **项目文件**：默认存储在 `我的文档/AI_Novel_Projects/` 目录下。
*   **配置文件**：存储在系统 `AppData` 目录下（`%APPDATA%/AI_Novel_Writer/`）。
*   **导出文件**：默认导出到项目文件夹内的 `export/` 子目录中。

## 3. 项目管理

### 3.1 新建项目
1.  点击主界面左侧的 **"新建项目"** 按钮。
2.  输入**小说名称**（建议使用中文或英文，避免特殊字符）。
3.  点击确定，系统会自动初始化项目环境，包括章节目录、大纲文件等。

### 3.2 打开项目
*   在主界面左侧的项目列表中，点击对应的项目卡片即可打开。
*   如果项目列表为空，请检查是否更改了默认工作区路径。

### 3.3 设置工作区
*   如果在安装后移动了项目文件，可以在设置中重新指定 **"工作区路径"**。
*   默认路径为：`C:\Users\用户名\Documents\AI_Novel_Projects`。

---

## 4. 写作流程

### 4.1 编辑大纲（核心步骤）
1.  进入项目后，点击左侧导航栏的 **"大纲/设定"**。
2.  在编辑器中输入你的小说大纲。
    *   **建议格式**：
        ```markdown
        # 书名：xxx
        ## 核心梗：...
        ## 角色小传：
        - 主角：...
        - 配角：...
        ## 分章大纲：
        ### 第1章：...
        ### 第2章：...
        ```
3.  点击 **"保存大纲"**。系统会自动分析大纲，提取"剧情圣经"（用于保持上下文一致性）。

### 4.2 生成章节
1.  点击左侧导航栏的 **"章节生成"**。
2.  **选择章节**：输入要生成的章节号（如 1）。
3.  **设置参数**：
    *   **指导建议**：可以输入本章的具体要求（如"多描写环境"、"通过对话推进剧情"）。
    *   **上下文记忆**：默认开启，系统会参考前文内容。
4.  点击 **"开始生成"**。
    *   右侧日志区会实时显示 AI 的思考过程（规划 -> 撰写 -> 润色 -> 审校）。
    *   生成完成后，内容会自动保存。

### 4.3 阅读与导出
1.  点击左侧导航栏的 **"阅读/导出"**。
2.  在左侧列表选择已生成的章节进行阅读。
3.  **导出功能**：
    *   点击右上角的导出按钮，支持 **TXT**、**Word**、**EPUB** 三种格式。
    *   导出文件将保存在 `我的文档/AI_Novel_Projects/<项目名>/export/` 目录下。
