# 常见问题（FAQ）

## 1）程序启动慢 / 页面一直在加载

可能原因：

- onefile EXE 首次启动需要解压依赖（正常）
- 端口被占用，服务启动后无法绑定端口

建议：

- 等待 30–60 秒
- 关闭占用端口的软件（或重启电脑）
- 查看 EXE 同目录的 `ai_novel_startup.log`（若存在）

## 2）浏览器没有自动打开

建议：

- 等待 10–30 秒后手动打开 `http://localhost:8501`（或其他提示端口）
- 若长时间无响应，查看 `ai_novel_startup.log`

## 3）API 验证失败

常见原因：

- Key 填错/过期
- 网络环境无法访问对应服务
- 短时间请求过多触发限流

建议：

- 更换 Key 后重新验证
- 稍等 1–5 分钟再试
- 检查网络代理/防火墙

## 4）找不到配置文件 / 配置丢失

说明：

- API Key 会加密保存到运行目录：
  - `.api_keys.enc`
  - `.encryption_key`
- 如果把 EXE 复制到新目录运行，会在新目录重新生成一套文件，配置互不影响

## 5）生成章节偏短（<3500 字）

说明：

- 项目已在保存前做字数校验，不足会自动扩写补足
- 如果仍偏短，通常是扩写环节被中断或输出被截断

建议：

- 查看运行日志，确认是否出现“扩写补足”提示
- 尝试切换更稳定的链路策略或主写模型

## 6）PowerShell 无法激活虚拟环境

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
.\.venv\Scripts\Activate.ps1
```

## 7）打包失败或提示“拒绝访问”

常见原因：

- `dist/AI_Novel_Writer.exe` 正在运行，导致无法覆盖写入

解决：

- 先关闭正在运行的程序再打包
